## 🏓 Ping Pong Scorekeeper (with Deuce)

Two‑player table tennis scorekeeper, built from Colt Steele's tutorial **plus my own deuce logic enhancement**.

### Features
- Set a winning score (5, 7, 11, etc.)
- **Deuce rule** (custom): when both players reach the winning score, the game continues until one leads by 2 points – "DEUCE" is shown, and the win condition is disabled until the lead is achieved.
- Reset button to start a new match.

### Tech
- HTML5, CSS3, vanilla JavaScript
- No frameworks – pure DOM manipulation
